# Sales-Performance-and-Customer-Insights-Analysis
A data analysis project exploring sales performance, product trends, and customer satisfaction across Nigeria's regional markets using transactional data from 2024 and 2025

## Overview

This project cleans, consolidates, and analyzes two years of sales records alongside a manager dataset to surface actionable insights on revenue, product performance, and customer experience.

---

## Objectives

- Clean and standardize raw sales data from 2024 and 2025
- Remove duplicate transactions using Transaction ID as a unique key
- Consolidate both datasets into a single source of truth
- Map transaction ratings to experience labels (Excellent → Very Poor)
- Assign regional managers to transactions based on state
- Answer key business questions:
  - Who was the top-performing manager in 2025?
  - Which are the top 2 products by total revenue?
  - Who are the top 5 sales representatives?
  - Which product has the highest average customer rating?
  - Which payment method is most associated with Excellent ratings?
